Halaman ini berisikan segalanya tentang situs web Tony.Permadi.id yang terdiri dari gagasan situs web ini, perangkat keras dan lunak yang digunakan, standar yang dipakai, validator dan pemeriksa, termasuk segala permasalahan pembuatannya.

Maksud Pembuatan

Gagasan membuat situs web ini tidak hanya sebagai media online untuk berekspresi diri, tetapi juga sebagai media pembelajaran kami untuk terus selalu mengasah kemampuan kami mendesain situs web dengan HTML, AMPHTML, CSS, dan skrip.

Itu sebabnya, kami telah membuat situs web tony.permadi.id ini sesederhana mungkin, tapi dengan teknik desain situs web terbaru sesuai standar terkini. Bila anda perhatikan dengan baik, mengakses situs web ini cukup cepat. Ya, kecepatan adalah prioritas nomor 1 di situs ini.

Target Pembaca

Situs web ini ditargetkan untuk semua orang di Indonesia, terutama mereka yang suka belajar komputer, internet, dan Bahasa Inggris. Kami di sini adalah guru-guru komputer dan Bahasa Inggris di Jakarta, Indonesia. Kami mengajar banyak siswa dari berbagai macam usia.

Setiap halaman web tony.permadi.id tersedia dalam Bahasa Indonesia dan Bahasa Inggris. Ya, situs web nya bilingual! Tentu, anda dapat menggunakan Google Translate untuk dapat menterjemahkan segala macam situs web, termasuk situs web tony.permadi.id ini. üòÉ

Perangkat Keras

Perangkat yang kami gunakan untuk membuat situs web ini biasa-biasa saja. Kami memakai Toshiba Dynabook Satellite B550 dengan Intel Core i7 M640, 2 core, 4 thread, berkecepatan 2,8 GHz, 8 GB RAM, layar 15,6 inci yang cukup membantu melihat segalanya dengan jelas.

Hard disk internal yang kami gunakan adalah Samsung 860 EVO 500GB solid-state drive (SSD) dan hard disk eksternal Seagate Backup Plus 4TB. Sedangkan perangkat portabel yang kami gunakan adalah Apple iPhone 6S Plus dengan browser web Safari nya dan perangkat portabel berbasis Android nya yaitu Samsung Galaxy Note 4 dengan Chrome web browser.

Perangkat Lunak

Sistem operasi yang kami gunakan adalah Manjaro Linux XFCE 19.0.2. Untuk menjaga kompatibilitas dengan rekan kerja, kami juga menggunakan Microsoft Windows 10. Editor utama yang kami gunakan untuk mengedit AMP (Accelerated Mobile Pages) HTML: NotepadQQ (Linux) & Notepad++ (Windows).

Kami sangat menyarankan pengunjung situs web ini menggunakan Google Chrome atau Mozilla Firefox karena keduanya mendukung PWA (Aplikasi Web Progresif) dengan baik. PWA, seperti situs tony.permadi.id ini, dapat membantu pengguna untuk tetap mengakses situs ini tanpa koneksi internet. Keren kan?

Alat Analisa

Kami telah memeriksa situs tony.permadi.id ini puluhan kali memakai PageSpeed Insight, juga GTMetrix, dan alat online lain yang dulu telah dikembangkan AOL: WebPageTest. Kami ingin pastikan situs web ini cukup cepat untuk bisa diakses dan mulai dibaca kurang dari 3 detik.

Kami mendapat angka sempurna 100 untuk output desktop, tetapi kurang 1 angka untuk output seluler. Ini disebabkan oleh JavaScript nya AMP. Waktu Berinteraksi di desktop: 0,5 detik, sedang di ponsel: 1,7 detik. Ini jauh lebih baik dari rerata waktu pemuatan situs pada umumnya: 10 detik untuk desktop dan 27 detik untuk ponsel, seperti yang telah diteliti oleh Backlinko tentang kecepatan situs-situs web.

Baik GTMetrix dan WebPageTest memberikan skor kinerja situs web level "A" untuk situs web ini. GTMetrix melaporkan PageSpeed & YSlow Skor "A" 100% dan "A" 98% + waktu pemuatan 450 milidetik, sedangkan WebpageTest telah memberikan skor Uji Kinerja dengan 4 "A" dan 1 "B" dengan waktu pemuatan web 1,4 detik.

Alat Validator

Kami memastikan bahwa situs tony.permadi.id sesuai dengan standar AMP dan PWA yang tepat. Itulah sebabnya Google AMP Test kami gunakan untuk memeriksa validitas halaman situs web ini, AMP Project Validator untuk memeriksa kode-kode AMP, dan juga tidak ketinggalan Google Lighthouse untuk PWA.

Tes AMP ini memiliki tiga pratinjau di mesin pencari Google karena situs tony.permadi.id memenuhi prasyarat beberapa tata letak hasil di mesin pencari Google, baik untuk desktop maupun ponsel. Namun, tidak dijamin bahwa tampilan aktualnya di Google akan sama.

Kami juga gunakan fitur Audit dari Google Chrome yang dinamakan Lighthouse untuk memeriksa Kinerja, Aksesibilitas, Praktik Terbaik, SEO, dan PWA situs web. Nampak di gambar di atas, situs tony.permadi.id telah mendapat "kembang api" untuk skor 100 poin.

Penyimpanan

Kami menyimpan HTML utama kami di Github nya Microsoft, sementara aset gambar beserta video disimpan dengan baik di Cloudinary. Untuk berjaga-jaga, kami menempatkan file dan aset tony.permadi.id di Firebase Google. Kesemuanya gratis dengan batasan tertentu.

Alat Online Lainnya

Untuk mengelola Domain Name System (DNS) web tony.permadi.id, kami gunakan Cloudflare yang mengelola DNS lewat pusat datanya di 200+ kota di seluruh planet ini. Cloudflare juga berkemampuan luar biasa menjadi server Jaringan Pengiriman Konten (CDN) yang cepat dangan caching dan kecepatan yang tinggi.

Dengan layanan komputerisasi tanpa server Cloudflare Workers, kami dapat memperketat keamanan situs tony.permadi.id. Hasil uji SSL dari Qualys SSL Labs mengkategorikan situs web tony.permadi.id sebagai situs web sangat aman dengan SSL berkelas "A" secara umum.
