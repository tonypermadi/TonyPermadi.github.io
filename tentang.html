Tentang Tony.Permadi.id

Halaman ini berisikan segala hal tentang situs web Tony.Permadi.id: gagasan situs web ini, perangkat keras dan lunak yang digunakan, standar yang dipakai, validator dan pemeriksa, termasuk segala permasalahan pembuatannya.

Maksud Pembuatan

Gagasan membuat situs web ini tidak hanya sebagai media online untuk berekspresi diri, tetapi juga sebagai media pembelajaran kami untuk terus selalu mengasah kemampuan kami mendesain situs web dengan HTML, AMPHTML, CSS, dan skrip.

Itu sebabnya, kami telah membuat situs web Tony.Permadi.id ini sesederhana mungkin, tapi dengan teknik desain situs web terbaru sesuai standar terkini. Bila anda perhatikan dengan baik, mengakses situs web ini cukup cepat. Ya, kecepatan adalah prioritas nomor 1 di situs ini.

Target Pembaca

Situs web ini ditargetkan untuk semua orang di Indonesia, terutama mereka yang suka belajar komputer, internet, dan Bahasa Inggris. Kami di sini adalah guru-guru komputer dan Bahasa Inggris di Jakarta, Indonesia. Kami mengajar banyak siswa dari berbagai macam usia.

Setiap halaman web Tony.Permadi.id tersedia dalam Bahasa Indonesia dan Inggris. Ya, situs web bilingual! Anda juga dapat menggunakan Google Translate untuk menterjemahkan situs web, termasuk situs web Tony.Permadi.id ini.

Perangkat Keras

Perangkat yang kami gunakan untuk membuat situs web ini biasa-biasa saja. Kami memakai Toshiba Dynabook Satellite B550 dengan Intel Core i7 M640, 2 core, 4 thread, berkecepatan 2,8 GHz, 8 GB RAM, layar 15,6 inci yang cukup membantu melihat segalanya dengan jelas.

Hard disk internal yang kami gunakan adalah Samsung 860 EVO 500GB solid-state drive (SSD) dan hard disk eksternal Seagate Backup Plus 4TB. Sedangkan perangkat portabel yang kami gunakan adalah Apple iPhone 6S Plus dengan browser web Safari nya dan perangkat portabel berbasis Android nya yaitu Samsung Galaxy Note 4 dengan Chrome web browser.

Perangkat Lunak

Kami menggunakan sistem operasi Manjaro Linux XFCE 21.2.6. Untuk tetap kompatibel dengan rekan kerja, kami juga menggunakan Microsoft Windows 10. Editor utama yang kami gunakan untuk mengedit kode HTML5 (HyperText Markup Language 5) adalah NotepadQQ (Linux) & Notepad++ (Windows).

Kami sangat menyarankan pengunjung situs web ini menggunakan Google Chrome atau Mozilla Firefox karena keduanya mendukung PWA (Aplikasi Web Progresif) dengan baik. PWA, seperti situs Tony.Permadi.id ini, dapat membantu pengguna untuk tetap mengakses situs ini tanpa koneksi internet. Keren kan?

Alat Analisa

Kami telah memeriksa situs Tony.Permadi.id ini puluhan kali memakai PageSpeed Insight, juga GTMetrix, dan alat online lain yang dulu telah dikembangkan AOL: WebPageTest. Kami ingin pastikan situs web ini cukup cepat untuk diakses dan mulai dibaca kurang dari 3 detik.

Google PageSpeed Insights memberikan skor sempurna 100 untuk tampilan seluler maupun desktop Tony.Permadi.id. Indeks Kecepatan situs web ini: 0,3 detik untuk desktop dan 1,0 detik untuk seluler. Ini sangat jauh lebih baik dibandingkan situs-situs web yang umumnya berkecepatan 11,4 detik dan 4,7 detik untuk desktop dan seluler. Ini berdasarkan pada data Backlinko tentang kecepatan situs-situs web.

Baik GTMetrix dan WebPageTest memberikan nilai kinerja tingkat "A" untuk situs web ini. GTMetrix Grade "A" dengan kinerja 100% dan waktu pemuatan halaman web kurang dari 0,4 detik, sedangkan WebpageTest melaporkan Indeks Kecepatan halaman web 0,391 detik.

Alat Validator

Kami pastikan bahwa situs Tony.Permadi.id sesuai dengan standar AMP dan PWA yang tepat. Itulah sebabnya Nu HTML Validator kami gunakan untuk memeriksa validitas halaman situs web ini, W3C CSS Validator untuk Cascading Style Sheets, JSHint untuk Javascripts, dan juga Google Lighthouse untuk komponen Aplikasi Web Progresif.

Nu HTML Checker memberikan validasi hijau untuk (X)HTML5 di Tony.Permadi.id karena kode HyperText Markup Language 5 yang valid + Accessible Rich Internet Applications (ARIA) + Scalable Vector Graphics (SVG) 1.1 + Math Markup Language (MathML) 2.0.

Layanan Validasi CSS W3C melaporkan bahwa situs Tony.Permadi.id juga valid untuk CSS 3 + SVG sehingga dapat memakai icon khusus CSS W3C untuk menunjukkan validasinya.

Kami juga gunakan fitur Audit dari Google Chrome yang dinamakan Lighthouse untuk memeriksa Kinerja, Aksesibilitas, Praktik Terbaik, SEO, dan PWA situs web. Nampak di gambar di atas, situs Tony.Permadi.id telah mendapat "kembang api" untuk skor 100 poin.

Penyimpanan

Kami menyimpan HTML utama kami di Github nya Microsoft, sementara aset gambar beserta video tersimpan dengan baik di Cloudinary. Untuk berjaga-jaga, kami juga tempatkan file dan aset Tony.Permadi.id di Firebase Google. Kesemuanya gratis dengan batasan tertentu.

Alat Online Lainnya

Untuk mengelola Domain Name System (DNS) Tony.Permadi.id, kami gunakan Cloudflare yang mengelola DNS lewat pusat datanya di 200+ kota di seluruh planet ini. Cloudflare juga berkemampuan luar biasa menjadi server Jaringan Pengiriman Konten (CDN) yang cepat dangan caching dan kecepatan yang tinggi.

Dengan layanan komputerisasi tanpa server Cloudflare Workers, kami dapat memperketat keamanan situs Tony.Permadi.id. Hasil uji SSL dari Qualys SSL Labs mengkategorikan situs web Tony.Permadi.id sebagai situs web sangat aman dengan SSL berkelas "A" secara umum.
