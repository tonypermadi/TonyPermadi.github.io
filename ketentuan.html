Pengantar

Syarat & Ketentuan ini adalah perjanjian antara Tony Permadi, disebut sebagai Operator Situs Web atau Kami, dan Anda, disebut sebagai Pengguna / Anda. Ini termasuk syarat, ketentuan, dan juga batasan penggunaan Tony.permadi.id, yang seterusnya akan disebut sebagai Situs Web atau Layanan.

Isi

Anda bertanggungjawab penuh atas akurasi, legalitas, kepantasan, dan hak-hak intelektual untuk menggunakan konten dan berkomentar. Anda mengizinkan kami untuk membagikan, menyalin, memformat, dan menampilkannya untuk memberikan layanan yang terbaik untuk Anda. Kami berhak menghapus konten atau komentar yang melanggar kebijakan kami.

Penyimpanan

Kami melakukan penyimpanan rutin situs web ini, isi dan komentarnya, dan akan melakukan yang terbaik untuk memastikan keakuratan penyimpanan ini. Jika terjadi kegagalan teknis perangkat keras, kami akan mengunggah data penyimpanan agar situs web dapat diakses.

Tautan Situs Web

Meski situs web ini bertautan dengan situs lain, tidaklah secara langsung/tidak langsung menyiratkan hubungan, dukungan, atau afiliasi apapun dengan situs web yang tertaut, kecuali dinyatakan secara khusus di sini. Afiliasi yang ada di situs web ini berarti jika Anda mengklik tautan dan membeli suatu barang, operator situs web ini akan menerima komisi afiliasi.

Kami tidak bertanggungjawab mengevaluasi dan menjamin penawaran bisnis atau individu apa pun atau konten situs web mereka. Kami juga tidak bertanggungjawab atas segala layanan situs-situs web tersebut. Anda harus hati-hati meninjau pernyataan hukum dan ketentuan lain dalam mengakses situs web lain yang terhubungkan. Akses ke situs web lain di luar situs ini adalah risiko Anda sendiri.

Larangan

Selain syarat dan ketentuan seperti yang diatur disini, dilarang mengakses situs web ini untuk:
- tujuan amoral atau melanggar hukum;
- meminta orang untuk melanggar hukum;
- melanggar peraturan dan hukum apa pun;
- melanggar hak kekayaan intelektual;
- mengirim informasi yang menyesatkan;
- mengunggah virus atau kode berbahaya;
- mengumpulkan informasi pribadi orang lain;
- melakukan spam, bom, phish, crawl, scrape;
- mengelak dari fitur keamanan situs web ini;
- melecehkan, menghina, mendiskriminasikan atas kelamin, agama, ras, asal, disabilitas.

Hak Intelektual

Perjanjian ini tidak mentransfer pada Anda hak intelektual operator situs web ini atau pihak ketiga. Merek dan logo di situs web ini adalah milik operator situs web. Merek dagang lain dapat merupakan merek dagang milik pihak ketiga lainnya. Akses Anda ke situs web dan layanan kami tidak memberi Anda hak untuk mereproduksi atau menggunakannya.

Privasi

Saat Anda berada di sini, server merekam info yang dikirim peramban Anda. Datanya dapat berupa info seperti Alamat IP, jenis perangkat, browser, sistem operasi, bahasa, halaman situs web yang dikunjungi, waktu yang telah Anda gunakan, beserta data statistik lainnya.

Anda dapat mengakses situs web ini tanpa memberitahu siapa Anda. Namun, jika Anda ingin menggunakan fitur situs web ini, Anda akan diminta untuk memberikan informasi pribadi seperti nama dan alamat email. Kami mendapatkan infonya saat Anda berkomentar atau mengisi formulir apa pun di situs web ini.

Kami menyimpan dan menggunakan informasi pribadi Anda untuk kebutuhan mematuhi hukum, menyelesaikan perselisihan, dan hal hukum lainnya sesuai yang berlaku. Kami dapat menggunakan data agregat apa pun yang berasal dari info pribadi Anda setelah Anda memperbarui ataupun menghapusnya.

Kami akan mengungkapkan informasi apapun yang telah kami kumpulkan jika diminta sesuai proses hukum, yang kami percaya beritikad baik bahwa pengungkapan diperlukan untuk melindungi hak-hak kami, Anda dan yang lainnya, menghindari penipuan, ataupun atas permintaan pemerintah Republik Indonesia.

Kuki Internet

Kami menggunakan potongan kecil data yang disebut "kuki internet" yang disimpan baik di peramban saat Anda mengakses situs web ini. Kuki digunakan untuk mengingat Anda dan preferensi yang Anda pakai saat berkunjung.

Kami gunakan kuki internet agar Anda mudah mengakses situs web ini. Kuki internet dapat memberitahukan tentang kunjungan Anda di situs web ini, yang datang dari area tertentu dan juga memiliki aktivitas di jejaring sosial.

Jika Anda tidak menyukai hal ini, Anda dapat mengatur browser Anda untuk menghapus kuki yang ada dan tidak menerima kuki baru. Silah kunjungi internetcookies.org untuk mempelajari lebih lanjut tentang kuki ini dan caranya untuk mengaktif/nonaktifkannya.
