Isi

Kami memiliki hak untuk menghapus konten atau komentar apa pun yang ada yang melanggar kebijakan kami.

Penyimpanan

Kami melakukan penyimpanan rutin situs web ini, isi dan komentarnya, dan akan melakukan yang terbaik untuk memastikan kelengkapan dan keakuratan penyimpanan ini. Jika terjadi kesalahan ataupun kegagalan perangkat keras, kami akan mengunggah penyimpanan untuk meminimalkan dampak dan waktu henti.

Tautan

Meski situs web ini bertautan dengan situs lain, kami tidak, secara langsung atau tidak langsung, menyiratkan hubungan, dukungan, atau afiliasi apa pun dengan situs web yang tertaut, kecuali dinyatakan secara khusus di sini. Beberapa tautan di situs web ini berupa "tautan afiliasi". Ini berarti jika anda mengklik tautan dan membeli suatu barang, operator situs web ini akan menerima komisi afiliasi.

Kami tidak bertanggungjawab mengevaluasi dan menjamin penawaran bisnis atau individu apa pun atau konten situs web mereka. Kami juga tidak bertanggungjawab atas segala layanan situs-situs web tersebut. Anda harus hati-hati meninjau pernyataan hukum dan ketentuan lain dalam mengakses situs web lain yang terhubungkan. Akses ke situs web lain di luar situs ini adalah risiko anda sendiri.

Larangan

Selain syarat dan ketentuan seperti yang diatur disini, dilarang mengakses situs web ini untuk:
- tujuan amoral atau melanggar hukum;
- meminta orang untuk melanggar hukum;
- melanggar peraturan dan hukum apa pun;
- melanggar hak kekayaan intelektual;
- mengirim informasi yang menyesatkan;
- mengunggah virus atau kode berbahaya;
- mengumpulkan informasi pribadi orang lain;
- melakukan spam, bom, phish, crawl, scrape;
- mengelak dari fitur keamanan situs web ini;
- melecehkan, menghina, mendiskriminasikan atas kelamin, agama, ras, asal, disabilitas.

Hak Intelektual

Perjanjian ini tidak mentransfer pada anda hak intelektual operator situs web ini atau pihak ketiga. Merek dan logo di situs web ini adalah milik operator situs web. Merek dagang lain dapat merupakan merek dagang milik pihak ketiga lainnya. Akses anda ke situs web dan layanan kami tidak memberi anda hak untuk mereproduksi atau menggunakannya.

Privasi

Saat anda berada di sini, server kami otomatis merekam info yang dikirim browser internet anda. Data-datanya dapat berupa info seperti alamat IP perangkatnya, jenis browser, sistem operasi, bahasa anda, halaman situs web yang dikunjungi, berapa banyak waktu yang telah anda gunakan, beserta data statistik lainnya.

Anda dapat mengakses situs web ini tanpa memberitahu siapa anda. Namun, jika anda ingin menggunakan fitur situs web ini, anda akan diminta untuk memberikan informasi pribadi seperti nama dan alamat email. Kami mendapatkan infonya saat anda berkomentar atau mengisi formulir apa pun di situs web ini.

Kami menyimpan dan menggunakan informasi pribadi anda untuk kebutuhan dapat mematuhi kewajiban hukum, menyelesaikan perselisihan, dan hal hukum lainnya sesuai yang berlaku. Kami dapat menggunakan data agregat apa pun yang berasal dari info pribadi anda setelah anda memperbarui ataupun menghapusnya.

Kami akan mengungkapkan informasi apa pun yang telah kami kumpulkan jika diminta sesuai hukum, seperti panggilan pengadilan ataupun proses hukum lain saat kami percaya dengan itikad baik bahwa pengungkapan diperlukan untuk melindungi hak-hak kami, anda dan yang lainnya, menghindari penipuan, atau pun atas permintaan pemerintah Republik Indonesia.

Cookie

Kami menggunakan data kecil yang disimpan dalam teks yang dikenal sebagai "cookie" yang tersimpan di browser anda saat mengakses situs web ini. Cookie ini digunakan sebagai pengingat kunjungan dan pengaturan situsnya.

Kami menggunakan cookie internet ini untuk membuat anda nyaman mengakses situs web ini. Cookie bisa memberitahu kami anda telah mengunjungi situs web ini, bahwa anda datang dari area tertentu di dunia ini, dan bisa juga memiliki aktivitas tertentu di jejaring sosial.

Jika anda tidak menyukai hal ini, anda dapat mengatur browser anda untuk menghapus cookie yang ada dan tidak menerima cookie baru. Silah kunjungi internetcookies.org untuk mempelajari lebih lanjut tentang cookie ini dan caranya untuk mengaktif/nonaktifkannya.
