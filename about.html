About Tony.Permadi.id

The page is all about Tony.Permadi.id website containing the idea of the website, the hardware and software used, the conforming standards, the online validators and checkers, including the problems we encountered during its creation.

The Idea

The idea of creating this website is not only to have an online media for us to express what is in our minds, but also as a learning media to keep us being updated with the internet web design technologies: the HTML5, CSS3, and the JavaScripts.

Hence, we have created this Tony.Permadi.id website as simple as possible, but with up-to-date website design techniques that conform to the current internet standards. You probably notice that it is quite fast accessing the pages here. Yes, speed is the number 1 priority here.

The Target

This website is intended to be read by the local Indonesians who love learning computers with its internet stuff, and the English language. We are a bunch of computers & English teachers here in the metro area of Jakarta, Indonesia, and we teach a lot of students of all ages.

Every web page of Tony.Permadi.id is available in both Bahasa Indonesia and English. Yes, it is a bilingual website! Of course you can use Google Translate to translate any web pages on the net, including Tony.Permadi.id website.

The Hardware

The core device we use for the creation of this website is not that great. We use a notebook computer of Toshiba Dynabook Satellite B550 with its Intel Core i7 M640 processor, 2 cores, 4 threads, clocked at 2.8 GHz, 8 GB RAM and 15.6" screen which helps us see things clearly.

The hard disk drive we use for internal storage is the Samsung 860 EVO 500GB solid-state drive (SSD) accompanied by an external hard disk drive of Seagate Backup Plus 4TB. While the portable devices we use to check the site are Apple iPhone 6S Plus with its Safari web browser and an Android-based portable device Samsung Galaxy Note 4 with Google Chrome.

The Software

The main operating system we use daily is Manjaro Linux XFCE 21.2.6. However, for compatibility reasons with our colleagues, we also use Microsoft Windows 10. The main source editors we use daily to edit the HTML5 (HyperText Markup Language 5) codes are NotepadQQ (Linux) & Notepad++ (Windows).

We strongly suggest all of our website visitors use Google Chrome or Mozilla Firefox as both of them support the PWA (Progressive Web Applications) very well. The PWA, which this Tony.Permadi.id website has been created for, can help the users to keep accessing this website with no internet connection. Cool, eh?

The Analyzers

We have checked this Tony.Permadi.id website tens of times using the PageSpeed Insight, the GTMetrix, and also the online tool originally developed by AOL: WebPageTest. We want to make sure that this website is fast enough for everyone to access it in under three seconds.

Google PageSpeed Insights gave a perfect 100 score for both the mobile and desktop display of Tony.Permadi.id. The website has the Speed Index of 0.3 seconds for the desktop and 1.0 second for the mobile. This is a lot better than most websites which have the average speed index of 11.4 seconds and 4.7 seconds for the desktop and mobile respectively. That is what Backlinko learned about page speed.

Both GTMetrix and WebPageTest gave a grade "A" level performance scores for this website. GTMetrix reported its GTMetrix Grade "A" with 100% performance and a loading time of less than 0.4 seconds, while the WebpageTest gave a similar report of 0.391 second Speed Index.

The Validators

We make sure that Tony.Permadi.id website conforms to the right standards of HTML5, CSS, Javascript and PWA. That is why we use Nu HTML Validator to check the validity of this website's HTML pages, the W3C CSS Validator for the Cascading Style Sheets, JSHint for the Javascripts, and also the Google Lighthouse for the Progressive Web App components.

The Nu HTML Checker gave a green (X)HTML5 validation result for Tony.Permadi.id as it has valid codes of HyperText Markup Language 5 + Accessible Rich Internet Applications (ARIA) + Scalable Vector Graphics (SVG) 1.1 + Math Markup Language (MathML) 2.0.

The W3C CSS Validation Service reported that Tony.Permadi.id website also validates as CSS Level 3 + SVG and could use a special W3C CSS icon to show its validation result.

We also use Google Chrome web browser's Audit feature called Lighthouse to check the website's Performance, Accessibility, Best Practices, SEO, and PWA. As shown here, Tony.Permadi.id website has been given the "fireworks" for its perfect 100-point scores.

The Storage

We put our main HTML pages on Microsoft's Github, while the image and video assets are stored well in Cloudinary. As a precautionary measure, we put Tony.Permadi.id website files and assets on Google's Firebase. All are free for the general public with certain limitations.

The Online Tools

To manage the Domain Name System (DNS) of Tony.Permadi.id site, we trust Cloudflare, Inc. We always feel comfortable using it as it does not only manage the DNS through its data centers across 200+ cities around this planet but also its great capabilities of being a fast Content Delivery Network (CDN) server.

With Cloudflare's serverless computing future of Cloudflare Workers, we tighten the security of Tony.Permadi.id website. Thanks to the free SSL Server Test provided by Qualys SSL Labs, it is confirmed that Tony.Permadi.id website is classified as an "A" class SSL-secured website.

As Vince Lombardi once said: "Perfection is not attainable, but if we chase perfection we can catch excellence."

So, pardon me, if this Tony.Permadi.id website is not performing well or not according to your preference. But we are all teachers here, and good teachers love learning & keep learning. Thank you for reading this About website page of Tony.Permadi.id. You have a good one!
